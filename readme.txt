== Description ==

The "Light Code Block" plugin is the simplest and lightest plugin for inserting and displaying code.

How to use:

* Install the plugin.
* Go to TinyMCE editor (on the posts page for example).
* Click the button labeled "LC" in the TinyMCE editor.
* Once the popup window opens, paste your code into it and click OK.
* Save the post.


== Installation ==

1. Install the "Light Code Block" plugin through the standard WordPress plugin installation page.
2. Or unzip and upload the "light-code-block" to the '/wp-content/plugins/' directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
